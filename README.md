# RadioWave

Landing UI para una estación de radio (React + TypeScript + Vite + Tailwind).

## Desarrollo

```bash
cd radio-wave
npm ci
npm run dev
```

## Build

```bash
cd radio-wave
npm run build
```

## Deploy en Vercel

El repositorio incluye un `vercel.json` en la raíz que compila `radio-wave/` y sirve `radio-wave/dist` como SPA.
